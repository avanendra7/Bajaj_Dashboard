# 📊 Bajaj Insurance Sales Dashboard

## 🔍 Overview
This project is a Power BI dashboard that shows insurance data like claim amount, premium amount, and sum assured. It helps in understanding trends and comparisons across regions, months, and genders.

## 🎯 Objectives
- Analyze claim amount, premium amount, and sum assured  
- Compare data by region and gender  
- Track monthly premium trends  
- Provide interactive filtering  

## 📌 Features
- KPI Cards:
  - Claim Amount: 53M  
  - Sum Assured: 210M  
  - Premium Amount: 6M  

- Region-wise Analysis  
- Monthly Premium Trends  
- Gender-based Comparison  
- Filters for Policy Type and Gender  

## 🛠 Tools Used
- Power BI  
- Data Visualization  

## 📈 Insights
- East and West regions have higher claims  
- Male claims are higher than female  
- Premium varies across months  

## 📂 Files
- Dashboard.pbix  
- Dataset (if available)  
- Screenshot  

## 🚀 How to Use
1. Download the .pbix file  
2. Open in Power BI Desktop  
3. Use filters to explore data
